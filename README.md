# PeopleTracking

There is 3 different projects in this repo. To open or work with any of them, follow these steps :
1. For the JavaScript project, open the folder with Visual Studio Code and open the index.html with Live Server extension.
2. For the Python project, open the folder with PyCharm and choose your python interpreter. (If needed you may have to install the missing libreries)
3. For the Service web (REST API) project, open the folder with IntelliJ IDEA.
